⚽ TOUCHLINE
Touchline é um jogo de gerenciamento de futebol em modo texto, inspirado em Football Manager, Elifoot e Brasfoot, desenvolvido em Python, com foco em simplicidade, simulação rápida e evolução incremental.

O projeto está atualmente em ALPHA, com sistemas centrais ainda em desenvolvimento.

📌 Estado do Projeto
Versão atual: 0.1.0-alpha
Status: Jogável (Alpha inicial)
Plataforma: CLI (Terminal)
Linguagem: Python 3.12+
⚠️ Quebras de compatibilidade podem ocorrer entre versões alpha.

🎮 Funcionalidades Atuais
Escolha de clube
Elencos gerados automaticamente
Jogadores separados por posição
Força individual e média do elenco
Simulação de temporada nacional simples
Classificação final do campeonato
🧠 Filosofia do Projeto
Simples antes de complexo
Jogável antes de completo
Clareza de código acima de otimização precoce
Touchline não busca realismo extremo neste estágio, mas sim uma base sólida para expansão futura.

🗺️ Roadmap (Planejado)
Próximas versões (resumido)
Competições internacionais
Sistema de divisões (Série B, acessos e rebaixamentos)
Persistência de temporadas (save/load)
Evolução e envelhecimento de jogadores
Mercado de transferências básico
Nada aqui é promessa de prazo — apenas direção.

🗂️ Estrutura do Projeto
Touchline/ ├── core/ # Lógica central (clubes, jogadores, simulação) ├── data/ # Dados fixos (clubes, nomes, sobrenomes) ├── ui/ # Interface em terminal ├── main.py # Ponto de entrada do jogo
